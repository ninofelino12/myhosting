Tahapan Membuat Flask Python dengan Env

1. Persiapan:

Pastikan Python terinstal: Minimal versi 3.6. Pastikan virtualenv terinstal: pip install virtualenv. Buat direktori proyek: mkdir my_flask_http. Masuk ke direktori proyek: cd my_flask_http.

2. Membuat Virtual Environment:

Buat virtualenv: python -m venv venv. Aktifkan virtualenv: source venv/bin/activate (Windows: venv\Scripts\activate). 3. Instal Flask dan Dependencies: Instal Flask: pip install Flask. Instal dotenv: pip install python-dotenv. Instal lainnya: Gunicorn, Werkzeug, Jinja2, dll. sesuai kebutuhan. 4. Buat File .env: Buat file .env: touch .env.
